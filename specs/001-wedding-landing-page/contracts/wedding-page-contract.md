# Wedding Page Contract

## Create or Update Wedding Page

### Request
- title
- coupleName
- weddingDate
- weddingTime
- featuredImage
- templateId
- status

### Response
- pageId
- publicUrl
- status
- validationErrors

## Public Page View

### Response
- page content rendered from the stored wedding event details
- featured image
- date and time display
- template styling
