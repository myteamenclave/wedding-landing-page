# Quickstart: Reusable Wedding Landing Page

## Prerequisites
- A browser with access to the application
- A valid account or admin access if editing is restricted

## Validate the Core Flow
1. Create a new wedding page.
2. Enter event details such as couple names, date, time, and upload an image.
3. Save and preview the page.
4. Publish the page and verify that it is viewable publicly.
5. Update the image or date/time and confirm the public page reflects the changes.

## Expected Outcome
- The page renders with the shared template structure.
- The updated event details appear correctly.
- Validation prevents incomplete or invalid submissions.
